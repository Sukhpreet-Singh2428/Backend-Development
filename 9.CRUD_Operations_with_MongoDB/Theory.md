# **mongoDB connection**

install mongoose package

`npm i mongoose`

* mongoose → mongoose helps in communication between node server to mongdb server.

> Mongoose is an ODM (Object Data Modeling) library that helps
> Node.js applications communicate with MongoDB.
> It provides schema structure, validation, and easy CRUD APIs.
